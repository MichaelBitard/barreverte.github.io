--- layout: post title: Faire du xml facilement en java avec XStream tags: - bruno - java - librairie - marshalling - unmarshalling - xml status: publish type: post published: true meta: _wp_old_slug: '' _edit_last: '2' --- Récemment nous avons du sérialiser des objets dans des fichiers, et une des personnes dans l'équipe a proposé XStream. C'est une librairie développée par ThoughtWorks (la société de Martin Fowler) qui permet de faire du marshalling/unmarshalling d'objet en XML. J'avais eu beaucoup de mauvaises expériences avec les librairies XML Java (soit trop intrusives niveau métier comme par exemple JAXB soit trop bas niveau comme DOM/SAX). XStream répond à toutes les attentes : 
public class MonObjet {
    private int monAttributEntier = 3;
    private String monAttributString = "attribut";

    public static void main(String[] args) {
        System.out.println(new XStream().toXML(new MonObjet()));
    }
}
Ce qui donne : 
   
     3
     attribut
